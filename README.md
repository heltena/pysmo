pysmo
=====

Python module to read/write/manipulate SAC (Seismic Analysis Code) files.

Build
-----

    sudo python setup.py build
    sudo python setup.py install


Documentation
-------------

The pysmo documentation is available at https://pysmo.readthedocs.io/
